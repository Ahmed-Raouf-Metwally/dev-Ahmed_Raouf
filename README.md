## üöÄ Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd dev-Ahmed_Raouf
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm start
    ```
    The app will run at `http://localhost:3000`.

4.  **Build for production**
    ```bash
    npm run build
    ```
